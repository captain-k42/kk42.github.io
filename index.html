皮皮直连 是一个用于传输文件、共享屏幕、视频会议、云剪贴板的网站，只需要在不同设备上打开皮皮直连网站，就能建立1对1、1对多连接，进行数据传输，非常方便。@Appinn

这是一款基于 WebRTC 的产品，来自发现频道。

皮皮直连可以很轻松的帮你在手机、电脑、iPad 等设备上建立一对一、一对多的直连通道，然后就可以实现传送大文件、同步剪贴板、共享屏幕桌面、视频会议等功能。

由于有浏览器就能用（现代浏览器），所以可以算是一款能让所有设备都可以用的，无需安装任何客户端的文件传输工具，不限于局域网，可以在很远的距离上进行传输文件，速度取决于共享双方的速度。

使用非常简单，打开皮皮直连网站，就会获得一个 ID 以及一个 URL 和二维码，连接方式也同样，输入 ID、打开 URL 或者扫码都行：

连接成功后，就可以在任意一段传输文件、文字、视频内容了，其他端会实时出现结果，只需要点击就能下载、复制：

用起来非常方便。不用了，只需要关闭浏览器即可，无负担。

感兴趣的同学可以前往官网使用。
